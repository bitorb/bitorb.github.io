Hugo 生成静态网站

2019-05-02 22:54:58
  • hugo

hugo 确实方便，可以专注于 markdown 写内容，模版负责布局与风格，与内容完全分离，生成全站网页基本秒出。

在 draft 模式下的开始写内容

将项目及子模块的最新的内容从远端同步到本地，然后以 draft 模式启动内置 webserver。

$ git pull origin 
$ git submodule update --remote
$ hugo server -D

deploy 到各个远端主机

github

先看一下要发布的内容有没有问题，然后退出 server 运行 hugo 命令生成网站。

$ hugo server
$ hugo
